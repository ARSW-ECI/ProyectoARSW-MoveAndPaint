WHAT IS MOVE-AND-PAINT?

It´s a game created by Carlos Iván Morera Gómez, Ricardo Andrés Pinto Rico and Carlos Alberto Ramírez Otero

MoveAndPaint is a multiplayer game in which 4 participants must paint the largest number of platforms composed of blocks and will gain who owns more for 40 seconds.

It´s deploy in Heroku: https://arsw-project.herokuapp.com/ and Amazon Web Services.
